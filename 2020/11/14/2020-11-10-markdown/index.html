[toc]

以下所有效果在 markdown 的typora编辑器和坚果云编辑器都能实现，但有些网站或编辑器不一定支持（如目录在github page 好像不支持）

关于换行

由于markdown编辑器的不同,可能在一行字后面，直接换行回车，也能实现换行，但是在Visual Studio Code上，想要换行必须得在一行字后面空两个格子才行

目录

直接在需要插入目录的地方输入下面语法：

[TOC]        在typora编辑器中大小写都行，但在GitHub pages不可用

会自动按标题生成目录，效果如下：

标题

语法如下（两种方法）：但在github pages 好像用一个#号没有使用二个##标题大啊

# 一级标题//注意在#号和标题直接要有空格，有的编辑器没有空格也能产生效果，应该只支持六级标题，往后不支持了
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题 
还有一种：在标题下加=或-
一级标题
====================
二级标题
===============
三级标题
------------------

字体

加粗

把要加粗的文字两端分别用两个*或_号包起来，语法如下：

**要加粗的字体**
__要加粗的字体__

要加粗的字体

要加粗的字体

斜体

把要使用斜体的文字两端分别用一个*或_号包起来，语法如下：

*斜体文字*
_斜体文字_

效果如下：

斜体文字

斜体文字

斜体加粗

把要倾斜和加粗的文字两端分别用三个*号或_包起来,语法如下：

***斜体加粗***
___斜体加粗___

效果如下：

斜体加粗

斜体加粗

删除线（中划线）

把要加删除线的文字两端分别用两个~~号包起来，语法如下：

~~要删除的文字~~

效果如下：

要删除的文字

下划线

把要加下滑线的文字两端用包起来（行内html写法），语法如下：

<u>下划线</u>

效果如下：

下划线

借助Latex公式实现上下线和下划线

语法如下：

$\overline{\text{上划线}}$
$\underline{\text{下划线}}$

效果如下：

$\overline{\text{上划线}}$

$\underline{\text{下划线}}$

高亮

把要高亮的文字两端分别用两个==号包起来，语法如下：

==高亮==

效果如下：

==高亮==

上标

把要设置上标的文字两端分别用一个^号包起来，语法如下：

X^2^

把要设置下标的文字两端分别用一个~号包起来，语法如下：

字体颜色和背景颜色设置

字体颜色（使用行内html）

<font color="#BC1717">想要设置颜色的文字</font>

想要设置颜色的文字

设置字体颜色同时设置字体加粗，语法如下：

<font color="#117113" style="font-weight:1000;">想要设置的字体颜色和字体加粗的文字!</font>

</div>

>>>> 四级引用
>>>>>>>> n级引用

有二种方法，语法如下

----------------             第一种方法，三个或三个以上“-”就行

插入图片

有二种方法：一种是行内式，一种是参考式

语法如下：

![图片名]（图片地址 "图片描述"）
例如![百度](https://www.baidu.com)或![图片](C:\Users\王新民\Pictures\分割线.PNG)

第二种：注意第二行可以放在文章的任何位置，当图片多时方便统一管理，标记可以是数组或字母如[1]或[abc],但两行的标记要保持一致

![图片][标记]
[标记]:C:\Users\王新民\Pictures\分割线.PNG "图片描述"

tips:可以使用html语法，对图片进行缩放，语法如下：
<img src="C:\Users\王新民\Pictures\标题.PNG" style="zoom:75%;" />

效果如下：

链接

链接和图片一样，有行内式和参考式两种，语法如下：

[可点击的文字](链接地址 "鼠标放在链接上出现的文字")      如[百度][https://www.baidu.com "百度"]

第二种：注意第二行可以放在文章的任何位置，当链接过多时，方便统一管理。标记可以是数组或字母如[1]或[abc],但两行的标记要保持一致

[可点击的文字][标记]
[标记]:链接地址 "鼠标放在链接上出现的文字" 


tips:使用html插入链接，并在新的标签页打开（target="_blank")，语法如下：
<a href="https://www.baidu.com" title="悬浮提示信息" target="_blank" rel="nofollow">百度</a>

效果如下：

第一种：行内式

百度

第二种：参考式

百度

自动链接

Markdown 支持以比较简短的自动链接形式来处理网址和电子邮件信箱，只要是用<>包起来， Markdown 就会自动把它转成链接

语法有两种：如下

<http://example.com>    第一种不显示<>

&lt;http://example.com&gt;   第二种github pages等不适用。特殊字符&lt;显示为<  字符&gt;显示为>  所以显示为<http://example.com>

效果如下：

http://example.com

<https://baidu.com/>

github API

列表

无序列表

在需要做成无序列表的每行文字前添加+或-或*，语法如下

- 情况1
+ 情况2
* 情况3

效果如下：

  • 情况1
  • 情况2
  • 情况3

有序列表

数字后加点，语法如下：注意在点与文字之间有个空格

1. 情况1
2. 情况2
3. 情况3

效果如下：

  1. 情况1
  2. 情况2
  3. 情况3

待办列表

语法如下：注意在-和[]之间有个空格，在[]和文字之间有个空格，[ ]内如果是空格是未勾选，如果是[x]表示勾选(x大小写都行)

-[ ] 未办事件
-[x] 完成事件

效果如下：

  • 未办事件

  • 完成事件

自定义列表

自定义型列表由名词和解释组成。一行写上定义，紧跟一行写上解释。解释的写法 一个冒号:紧跟一个缩进(Tab)或4个空格(有些不支持这种语法)

语法如下：

markdown
:    内容1
java
:    内容1
:    内容2
markdown
: 内容1
java
: 内容1
内容2

表格

语法如下：

|表头1|表头2|表头3|
|:--:|:--:|:--:|
|内容1|内容2|内容3|
|内容4|内容5|内容6|

第二行是为了分割表头和内容
第二行中的“-”，有一个就行了，可以多加几个对齐，好看。
第二行中的“：” 是控制内容和表头是否居中，不加两边的：默认居左
两边都加：表示居中，在左边添加：表示居左，在右边添加：表示居右

效果如下：

表头1 表头2 表头3
内容1 内容2 内容3
内容4 内容5 内容6

代码

单行代码

把单行代码的两端分别用单个 ` 包裹起来语法如下：

`代码内容`

效果如下：

代码内容

代码块

如果是多行代码，代码块开头结尾分别用三个```包裹起来，语法如下：

代码行1
代码行2
代码行3

效果如下：

代码行1
代码行2
代码行3

锚点

锚点分两步：第一步设置锚点（转到的位置），第二步设置要点击的按钮（点击这个按钮，就跳转到第一步的位置），语法如下：

<a name="锚点标记">锚点描述</a>    这是第一步，通常锚点描述省略，即<a name="锚点标记"></a>
<a href="#锚点标记">跳转描述</a>   这是第二步，注意跳转描述不能省略，不然没有点击按钮了

#号后面的锚点标记要和name后面的锚点标记一直

效果如下：

跳转到代码块处

脚注

脚注和锚点有点相似，分两步：第一步设置脚注标记，第二步设置脚注的解释内容和位置（一般都放在文章的末尾）

语法如下：

一段文字[^脚注标记]                第一步设置脚注标记

[^脚注标记]:对脚注的解释内容     第二步设置脚注的解释内容和位置

效果如下：

爱因斯坦[^1]是一个伟大的物理学家

缩写

语法如下：分两步：第一步：定义缩写词，第二步使用缩写词。缩写词的定义可以放在使用缩写词后面。

使用缩写词                        使用缩写词

*[缩写词]:缩写词的全部拼写            定义缩写词

例如

5G的名字是NR。

*[NR]:NEW RADIO

效果如下：(坚果云支持，typora好像不支持)

5G的缩写是：NR

*[NR]:NEW RADIO

转义字符

有些字符直接打印在屏幕上时，不代表本身含义，如：+ 加号 - 等，如果直接写+ 它会显示无序列表，而不是+，这时需要转义字符\

语法如下：

\要表示的字符    如要打印\ 则应该写 \\(第一个\把第二个\转义为\)
 如\+  \把+转义为+ 而不是无序列表

效果如下：

\

+
